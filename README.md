# Evermos API - Virtual Internship

Backend service sederhana untuk platform penjualan reseller Evermos.
Dibuat menggunakan Golang + MySQL + JWT Authentication.

---

## Tech Stack

- **Go (Golang)**
- **MySQL + GORM**
- **JWT Authentication**
- **Swagger API Documentation**
- **Gorilla Mux Router**
